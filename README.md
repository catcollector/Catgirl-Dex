# Catgirl-Dex

Fetches catgirls, missing settings.py to protect token.


Add settings.py:

import os

COMMAND_PREFIX = "!"

BOT_TOKEN = ""

NOW_PLAYING = COMMAND_PREFIX + "commands"

BASE_DIR = os.path.dirname(os.path.realpath(__file__))
